This research-paper recommendation system aims to help the researcher in his bibliographical search by being an alternative to traditional search engines. It recommends papers on the basis of a selection of papers proposed by the user. In contrast to conventional recommendation systems which are based on a one-shot interaction scenario, this system can work iteratively by using precedent recommendations as inputs for the next ones. The workflow is composed of four parts :

  1. Loading some papers as input to the recommendation engine, either by text search with Search or by loading from bibtex format records with Load.
  2. Launch the recommendation engine with the input papers and the chosen method with Recommend.
  3. Exploit the obtained recommendations, either by consulting the paper list with Exploit, or by getting some insight about the field with Dashboard.
  4. Save some results and export them in bibtex format with Save.

This is yet a prototype, so there might still be a few bugs and some parts can be slow ! This is especially true for the recommend methods which involve big database queries. In these cases, you may encounter 502 or 504 error but a retry should solve the problem.

User study

If you have some time and wish to contribute to the development of this tool, do not hesitate to take this user study.

Objectives of the study:

  • Assess the general interest of this tool for literature research.
  • Assess 2 recommendation methods according to various criteria.
  • Assess the ability of sorting features to improve displayed results according to various criteria.

Instructions for the user study:

  1. Select 1-5 papers about a research topic of interest as an input for recommendation (with Search and/or Load modules).
  2. Select a recommendation method and launch the recommendation process (Recommend module).
  3. Take a look at the first 10 results displayed (Exploit module).
  4. Use the sort features to improve the first 10 results.
  5. Take a look a the Dashboard module.
  6. Fill the survey form.
  7. (Optional, but highly appreciated) Repeat steps 1-6 with another untried recommendation method.
